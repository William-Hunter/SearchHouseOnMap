# 58 租房搜索
----------------------------

在实验楼看到一个很有趣的项目：[高德API+Python解决租房问题](https://www.shiyanlou.com/courses/599/labs/1978/document)
决定按照自己掌握的技术实现一遍，

### 思路
-----------------
从58同城抓取租房信息，持久化的存储在数据库里面，然后在网页里调用高德的API，创建一个电子地图，当用户选择一个地点，并且输入房租价位与搜索半径后，在数据里搜索符合条件的记录，然后组织成一个个的高德标记。
